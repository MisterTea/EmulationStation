    ESMAME

    ESMAME is a fork of EmulationStation that is designed to make playing arcade and console games on MAME extremely simple.

    Why make ESMAME?

    MAME is an AMAZING emulator, but many people don't use it because (1) it is too hard to configure and (2) it isn't obvious that MAME supports consoles.

    Since MAME is a frustrating experience, people have turned to emulator containers like Retroarch that try to wrap many emulators. MAME supports almost all of the systems that Retroarch supports (minus PSP and real-time N64) in a single binary that only needs to be configured once.

    How to install

    1. Download ESMAME from here.
    2. Open the ESMAME folder
    3. Get mame from here. Make sure you download the 64-bit version if you are running a 64-bit OS.
    4. Get Roms from Pleasuredome and put into the Roms folder
      • Tested using the "MAME 0.166 ROMs" torrent
      • There should be no sub-directories: all of the 7z files go in the Roms folder
    5. Get software from Pleasuredome and put into the Software folder
      • Tested using the "MAME 0.166 Software List ROMs" torrent
      • There should be a sub-folder for each machine and each sub-folder should contain 7z files (e.g. nes/*.7z)
    6. Get extras from Pleasuredome and put into the Extras folder
      • Tested using the "MAME 0.165 EXTRAs" torrent
      • there should be cheat.7z in the folder, along with several other folders and files

    How to run

    • run launch.bat
    • Wait until the ES frontend starts up
    • Change systems with left-right arrow keys, and select a system with 'a'
    • Select a game with 'up'/'down' or faster by holding 'd'/'c'
    • Go back to the main menu with 'z'

    How to set up a joystick

    • Enter settings with 'enter'
    • Press the down arrow to go to 'Configure Input' and press 'a'
    • Hold any button on your joystick to begin the configuration process
    • The 6-button inputs (top-left and top-right) and the shoulder buttons (L1/R1/L2/R2/L3/R3) are exclusive, so you can map them to the same buttons if appropriate.

    How to get game art

    • Enter settings with 'enter'
    • Press 'a' to enter Scraper
    • Choose 'Scrape Now'
    • Press the down arrow until you reach the START button and press 'a'
    • Wait.
